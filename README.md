# Organic E-commerce Website

This is a fully responsive e-commerce website built with Next.js, designed to sell organic products such as ghee, gud, and pulses. The application features a backend powered by MongoDB for managing stock and orders, ensuring a seamless shopping experience.

## Features

- **Responsive Design**: The website is optimized for various devices, providing a smooth user experience on desktops, tablets, and mobile phones.
- **Product Management**: Admins can easily manage products, including adding, updating, and deleting items from the inventory.
- **Shopping Cart**: Users can add products to their cart, update quantities, and proceed to checkout.
- **Checkout Process**: A streamlined checkout process that collects user information and processes orders efficiently.
- **API Integration**: The application includes RESTful API endpoints for managing products and orders, ensuring a robust backend.

## Technologies Used

- **Next.js**: A React framework for building server-side rendered applications.
- **MongoDB**: A NoSQL database for storing product and order information.
- **TypeScript**: For type safety and better development experience.
- **Tailwind CSS**: A utility-first CSS framework for styling the application.

## Getting Started

### Prerequisites

- Node.js (version 14 or higher)
- MongoDB (local or cloud instance)

### Installation

1. Clone the repository:
   ```
   git clone <repository-url>
   cd organic-ecommerce
   ```

2. Install dependencies:
   ```
   npm install
   ```

3. Set up environment variables:
   Create a `.env.local` file in the root directory and add your MongoDB connection string:
   ```
   MONGODB_URI=<your-mongodb-connection-string>
   ```

4. Run the development server:
   ```
   npm run dev
   ```

5. Open your browser and navigate to `http://localhost:3000`.

## Usage

- Browse products on the homepage and view details on individual product pages.
- Add items to your cart and proceed to checkout.
- Admins can access the admin dashboard to manage products and orders.
